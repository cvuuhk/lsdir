#  功能参考 

| short |      long      |            comment             |
| :---: | :------------: | :----------------------------: |
|   a   |    all-most    | 显示所有文件，除了 `.` 和 `..` |
|   i   |     inode      |           显示 inode           |
|   h   | human-readable |    以合适的单位显示文件大小    |
|   l   |                |          显示详细信息          |
|  `I`  | ignore=PATTERN |    忽略正则表达式匹配的文件    |
|   s   |      size      |         显示文件字节数         |
